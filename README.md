# VibraniumCode — Web Site

Sitio web oficial de VibraniumCode, empresa de desarrollo de software de Buenos Aires, Argentina.

🌐 **[vibraniumcode.com](https://vibraniumcode.com)**

---

## 🚀 Stack

- **Frontend** — Vue 3 + TypeScript + Vite
- **Estilos** — Tailwind CSS 4
- **Routing** — Vue Router 4
- **Backend** — Vercel Functions + Nodemailer
- **Deploy** — Vercel
- **Analytics** — Google Analytics 4

## 🛠️ Instalación

```bash
# Clonar el repo
git clone https://github.com/vibraniumCode/web-site-vc.git

# Entrar al proyecto
cd web-site-vc

# Instalar dependencias
npm install

# Correr en desarrollo
npm run dev

# Build para producción
npm run build
```
